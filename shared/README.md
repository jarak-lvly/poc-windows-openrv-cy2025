# Shared

This directory is bind-mounted into the Windows build environment using the standard [dockur/windows](https://github.com/dockur/windows) shared-folder mechanism.

It is intentionally empty. Users may use it to transfer build artifacts between the Windows build environment and the Linux host.

See the "How do I share files with the host?" [dockur/windows](https://github.com/dockur/windows) documentation for additional information.
